此项目是活动预约系统使用springboot + vue3写的
``使用方法``
先运行后端再运行前端   
后端
直接使用idea导入demo文件夹中的pom.xml文件  jdk17以上
前端使用cmd打开前端文件夹的根目录  然后执行npm instll安装npm
执行npm run dev运行整个前端项目     node版本号为v20.14.0
